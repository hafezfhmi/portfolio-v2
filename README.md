# portfolio v2

My portfolio website v2

## Tech Stack

**Client:** Next.js, TailwindCSS

## Run Locally

Clone the project

```bash
  git clone https://github.com/hafezfhmi/portfolio-v2.git
```

Go to the project directory

```bash
  cd portfolio-v2
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
